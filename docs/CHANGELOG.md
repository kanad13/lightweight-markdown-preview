# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.0] - 2025-11-02

### Added
- **Improved Build System:** Webpack bundling for optimized distribution
- **Enhanced Code Quality:** ESLint configuration update for consistent code style
- **Better Development Experience:** Streamlined development documentation and setup process

### Changed
- Updated dependencies to latest stable versions (marked, @types/node, eslint, @vscode/vsce, @vscode/test-electron)
- Repository structure reorganization for improved clarity and maintainability
- Enhanced activation and logging mechanisms

## [0.3.0] - 2025-01-28

### Added
- **MathJax Support:** LaTeX equation rendering with full math notation support
- **Mermaid Diagram Support:** Dynamic diagram and flowchart rendering
- **Syntax Highlighting:** Integrated highlight.js for improved code block appearance
- **Privacy-First Design:** Marketing and positioning as a privacy-friendly alternative

### Changed
- Refactored string quoting for consistency across codebase
- Optimized preview styling and presentation
- Consolidated developer documentation for better maintainability

### Fixed
- Improved markdown fence handling in examples
- Enhanced mermaid and math block extraction and restoration logic
